# chn-site-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面作为静态资源存放，不针对任何特定的域名或具体网站，仅用于资料整理与展示。

## 项目简介

`chn-site-guide` 是一个静态页面集合仓库，旨在以 HTML 格式保存和共享一些独立的信息页面。所有页面均为自包含的 HTML 文件，便于直接浏览或嵌入其他环境。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example.html    # 示例页面（说明用）
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── styles.css
│   └── ...
└── README.md           # 本文件
```

- **`pages/`**：核心目录，每个 HTML 文件对应一个独立页面。
- **`assets/`**：存放页面引用的样式、脚本或图片等公共资源。
- **`index.html`**：可选入口页，可用于导航或说明。

## 页面归档说明

- 每个页面文件应为完整、自包含的 HTML 文档。
- 页面之间无依赖关系，可独立打开和浏览。
- 建议在页面内添加必要的说明或元信息，以便理解内容用途。
- 归档内容不涉及特定网站、域名或商业推广信息。

## 维护说明

- 本仓库欢迎提交新的 HTML 页面或改进现有页面。
- 提交前请确保页面内容清晰、结构合理，不违反法律法规及平台政策。
- 建议通过 Pull Request 提交更改，并在描述中简要说明新增或修改的内容。
- 如有疑问或建议，可提交 Issue 进行讨论。
